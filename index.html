delete /teams/{teamId}/memberships/{userId}
(teamsTeamIdMembershipsUserIdDelete)
Remove the user's membership of the team.

Path parameters

userId (required)
Path Parameter — The Invotra UUID of the user.
teamId (required)
Path Parameter — The Invotra UUID of the team.

Responses

200

The user has been removed from the team.

404

The user or team does not exist, or the user was not a member of the team.

put /teams/{teamId}/memberships/{userId}
(teamsTeamIdMembershipsUserIdPut)
Update information about a user's team membership.

Path parameters

userId (required)
Path Parameter — The Invotra UUID of the user.
teamId (required)
Path Parameter — The Invotra UUID of the team.

Query parameters

user_uuid (required)
Query Parameter — Invotra UUID of the user to add to the group
role_type (required)
Query Parameter — default: member

Responses

200

The user's membership of this team was updated.

201

The user was added as a member of the team.

404

The user or team does not exist.
